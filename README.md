# 🛍️ ModernWear E-Commerce Website

A modern, responsive multi-page e-commerce website built using **HTML, CSS, and JavaScript**.  
This project is designed for beginners to understand real-world front-end development concepts including layout design, responsive UI, and interactive components.

## 📌 Project Overview

ModernWear is a fully responsive e-commerce front-end website that includes multiple sections such as product listings, banners, navigation, newsletter, and footer.

It demonstrates how a real-world shopping website is structured and styled using modern front-end techniques.

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Media Queries)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts
